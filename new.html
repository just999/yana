Membuat Tautan dalam HTML

Tautan atau hyperlink adalah koneksi dari satu sumber daya web ke sumber daya web lainnya. Tautan memungkinkan pengguna untuk berpindah dengan mudah dari satu halaman ke halaman lain, di server mana pun di seluruh dunia.

Tautan memiliki dua ujung, yang disebut jangkar. Tautan dimulai dari jangkar sumber dan mengarah ke jangkar tujuan, yang dapat berupa sumber daya web apa pun, misalnya, gambar, klip audio atau video, berkas PDF, dokumen HTML, atau elemen di dalam dokumen itu sendiri, dan sebagainya.

Secara default, tautan akan muncul seperti berikut di sebagian besar browser:

  • Tautan yang belum dikunjungi diberi garis bawah dan berwarna biru.

  • Tautan yang telah dikunjungi diberi garis bawah dan berwarna ungu.

  • Tautan yang aktif digarisbawahi dan berwarna merah.

Namun, Anda dapat menimpanya menggunakan CSS. Pelajari lebih lanjut tentang gaya tautan .

Sintaks Tautan HTML

Tautan ditentukan dalam HTML menggunakan <a>tag.

Tautan atau hyperlink dapat berupa kata, sekelompok kata, atau gambar.

<a href =" url "> Teks tautan </a>

Apa pun yang berada di antara tag pembuka <a>dan tag penutup </a>akan menjadi bagian tautan yang dilihat dan diklik pengguna di peramban. Berikut beberapa contoh tautannya:

Contoh

Coba kode ini »

<a href="https://www.google.com/">Google Search</a>
<a href="https://www.tutorialrepublic.com/">Tutorial Republic</a>
<a href="images/kites.jpg">
    <img src="kites-thumb.jpg" alt="kites">
</a>

Atribut ini hrefmenentukan target tautan. Nilainya dapat berupa URL absolut atau relatif.

URL absolut adalah URL yang mencakup setiap bagian dari format URL, seperti protokol, nama host, dan jalur dokumen, misalnya, https://www.google.com/, https://www.example.com/form.php, dst. Sementara itu, URL relatif adalah jalur relatif terhadap halaman, misalnya, contact.html, images/smiley.png, dst. URL relatif tidak pernah menyertakan http://awalan https://.

Menetapkan Target untuk Tautan

Atribut ini targetmemberi tahu peramban tempat membuka dokumen tertaut. Ada empat target yang ditentukan, dan setiap nama target diawali dengan _karakter garis bawah ( ):

  • _blank— Membuka dokumen yang ditautkan di jendela atau tab baru.

  • _parent— Membuka dokumen tertaut di jendela induk.

  • _self— Membuka dokumen tertaut di jendela atau tab yang sama dengan dokumen sumber. Ini adalah nilai default, sehingga tidak perlu menentukan nilai ini secara eksplisit.

  • _top— Membuka dokumen yang ditautkan di jendela browser penuh.

Cobalah contoh berikut untuk memahami cara kerja target tautan pada dasarnya:

Contoh

Coba kode ini »

<a href="/about-us.php" target="_top">About Us</a>
<a href="https://www.google.com/" target="_blank">Google</a>
<a href="images/sky.jpg" target="_parent">
    <img src="sky-thumb.jpg" alt="Cloudy Sky">
</a>

Kiat: Jika laman web Anda ditempatkan di dalam iframe, Anda dapat menggunakannya target="_top"pada tautan untuk keluar dari iframe dan menampilkan laman target di jendela peramban penuh.

Membuat Penanda Jangkar

Anda juga dapat membuat jangkar penanda untuk memungkinkan pengguna melompat ke bagian tertentu di halaman web. Penanda sangat berguna jika halaman web Anda sangat panjang.

Pembuatan bookmark merupakan proses dua langkah: pertama tambahkan idatribut pada elemen yang ingin dilompati, lalu gunakan idnilai atribut yang diawali tanda pagar ( #) sebagai nilai hrefatribut tag <a>, seperti ditunjukkan dalam contoh berikut:

Contoh

Coba kode ini »

<a href="#sectionA">Jump to Section A</a>
<h2 id="sectionA">Section A</h2>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit...</p>

Kiat: Anda bahkan dapat melompat ke bagian halaman web lain dengan menentukan URL halaman tersebut beserta jangkar (misalnya #elementId) dalam hrefatribut, misalnya, <a href="mypage.html#topicA">Go to TopicA</a>.

Membuat Tautan Unduhan

Anda juga dapat membuat tautan unduhan berkas dengan cara yang sama seperti menempatkan tautan teks. Cukup arahkan URL tujuan ke berkas yang ingin Anda unduh.

Dalam contoh berikut kami telah membuat tautan unduhan untuk file ZIP, PDF, dan JPG.

Contoh

Coba kode ini »

<a href="downloads/test.zip">Download Zip file</a>
<a href="downloads/masters.pdf">Download PDF file</a>
<a href="downloads/sample.jpg">Download Image file</a>
Catatan: Ketika Anda mengeklik tautan yang mengarah ke berkas PDF atau gambar, berkas tersebut tidak langsung terunduh ke hard drive Anda. Berkas tersebut hanya akan terbuka di peramban web Anda. Selanjutnya, Anda dapat menyimpan atau mengunduhnya ke hard drive Anda secara permanen.
